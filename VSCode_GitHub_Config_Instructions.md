# VSCode GitHub Config Instructions

1. Open a PowerShell Terminal

<img src="https://learn.microsoft.com/en-us/powershell/docs-conceptual/learn/ps101/media/figure1-1.jpg" width="300"/>

2. Apply the following commands, replacing `your@school.email` with your GitHub email address and replacing `username` with your GitHub username.

```bash
git config --global user.email "your@school.email"
git config --global user.name "username"
```
Once these steps have been followed you can commit, push and pull from VSCode
